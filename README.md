# cartoascii

![demo](https://i.imgur.com/SlaLIx5.png)

Google Maps for the Terminal (based on OpenStreetMap), packed with features!

### install
This requires Python3

```bash
git clone [https://github.com/yourusername/cartoascii](https://github.com/yourusername/cartoascii)
cd cartoascii
pip install -r requirements.txt
python main.py
```
